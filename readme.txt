# Speedy avo setup
example of 3/3 video wall in mode 2(16x16)pixels

1: Patch all 288 RGB segments by setting the quantity to 288 (Auto overlap universes :D)
2: Make or select the premade group and go to pixel mapping, and edit group layout.
3: click arrange and then set the width and height to what the jar-file outputs. i.e. 12x24
4: Once it has been arranged, select the groups of RGB fixtures from the edit group layouts page to make new groups
5: make all the groups of each panels you want.
6: profit :D
